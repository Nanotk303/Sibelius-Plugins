# Sibelius Plugins

Collection de plug-ins Sibelius developpes ensemble.

## Plug-ins

- [PCSet-remap](PCSet-remap/README.md): remappe les notes d'une selection vers les classes de hauteurs choisies les plus proches.
- [Expand-Compress-intervals](Expand-Compress-intervals/README.md): dilate ou compresse les intervalles d'une selection selon un ratio.

Chaque plug-in vit dans son propre dossier avec sa source ManuScript, son fichier `.plg` installable, sa documentation et ses tests.
